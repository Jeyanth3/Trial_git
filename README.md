# Trial Git Project

A collection of simple HTML pages and stylesheets.

## Project Structure

```
Trial_git/
├── index.html          # Main welcome page
├── name.html           # Simple name input form styled with Tailwind
├── twomangoonstone.html# Different versions per branch (yellow/orange mangoes)
├── style.css           # Global stylesheet with custom rules
├── Helloworld.txt      # Additional text file
└── README.md           # Project documentation
```

## Usage

Open any `.html` file in your web browser to view the pages.
